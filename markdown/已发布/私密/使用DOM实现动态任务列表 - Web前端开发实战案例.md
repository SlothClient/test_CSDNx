# 使用DOM实现动态任务列表 - Web前端开发实战案例
## 一、前言DOM（Document Object Model）是前端开发的核心技术之一，它如同连接HTML文档与JavaScript的桥梁。本文将通过一个完整的动态任务列表案例，演示如何利用原生JavaScript操作DOM元素，实现常见的增删改查功能。这个案例将涵盖事件处理、元素创建、样式操作等核心知识点，适合前端新手学习和实践。## 二、案例效果预览最终实现的功能包括：1. 添加新任务（回车确认）2. 标记完成状态（点击切换样式）3. 删除单个任务4. 清空全部任务5. 动态数据统计6. 本地存储持久化[外链图片转存失败,源站可能有防盗链机制,建议将图片保存下来直接上传(img-w9sCIkHb-1738511674557)(https://via.placeholder.com/600x400)]## 三、完整代码实现### HTML结构```html<div class="container">  <h1>任务管理器</h1>  <div class="input-group">    <input type="text" id="taskInput" placeholder="输入新任务...">    <button id="clearAll">清空全部</button>  </div>    <div class="stats">    总计: <span id="total">0</span>     已完成: <span id="completed">0</span>  </div>  <ul id="taskList"></ul></div>```### CSS样式```css.container {  max-width: 600px;  margin: 2rem auto;}.task-item {  display: flex;  align-items: center;  padding: 12px;  margin: 8px 0;  background: #f5f5f5;  border-radius: 4px;  transition: all 0.3s;}.completed {  background: #e0ffe0;  text-decoration: line-through;}.delete-btn {  margin-left: auto;  color: #ff4444;  cursor: pointer;}```### JavaScript核心逻辑```javascript// DOM元素缓存const taskInput = document.getElementById('taskInput');const taskList = document.getElementById('taskList');const clearBtn = document.getElementById('clearAll');// 初始化本地存储let tasks = JSON.parse(localStorage.getItem('tasks')) || [];// 初始化渲染renderTasks();// 添加任务事件taskInput.addEventListener('keypress', (e) => {  if (e.key === 'Enter' && taskInput.value.trim()) {    addTask(taskInput.value.trim());    taskInput.value = '';  }});// 清空全部事件clearBtn.addEventListener('click', clearAllTasks);// 任务列表事件委托taskList.addEventListener('click', (e) => {  const target = e.target;  const taskItem = target.closest('.task-item');    if (target.classList.contains('delete-btn')) {    deleteTask(taskItem.dataset.id);  } else if (taskItem) {    toggleComplete(taskItem.dataset.id);  }});// 添加任务函数function addTask(text) {  const newTask = {    id: Date.now(),    text,    completed: false  };    tasks.push(newTask);  updateStorage();  renderTasks();}// 删除任务函数function deleteTask(id) {  tasks = tasks.filter(task => task.id !== Number(id));  updateStorage();  renderTasks();}// 渲染函数function renderTasks() {  taskList.innerHTML = '';    tasks.forEach(task => {    const li = document.createElement('li');    li.className = `task-item ${task.completed ? 'completed' : ''}`;    li.dataset.id = task.id;        li.innerHTML = `      ${task.text}      <span class="delete-btn">×</span>    `;        taskList.appendChild(li);  });  updateStats();}// 更新统计数据function updateStats() {  document.getElementById('total').textContent = tasks.length;  document.getElementById('completed').textContent =     tasks.filter(task => task.completed).length;}// 持久化存储function updateStorage() {  localStorage.setItem('tasks', JSON.stringify(tasks));}```## 四、关键技术解析1. **事件委托优化**通过`closest()`方法实现事件委托，避免为每个任务单独绑定事件，显著提升性能。这种方式尤其适合动态生成的元素。2. **数据驱动视图**采用MVC模式，以tasks数组作为数据源，通过renderTasks()函数统一渲染，保持数据与视图同步。3. **本地存储集成**使用localStorage实现数据持久化，通过JSON序列化存储复杂数据结构。4. **现代DOM操作**- 使用`dataset`存储自定义数据- 利用`classList`操作CSS类- 采用模板字符串构建HTML## 五、功能扩展建议1. **编辑功能**```javascriptfunction enableEdit(taskItem) {  const textSpan = taskItem.firstChild;  const input = document.createElement('input');  input.value = textSpan.textContent;    taskItem.replaceChild(input, textSpan);  input.focus();    input.addEventListener('blur', () => {    updateTaskText(taskItem.dataset.id, input.value);  });}```2. **动画优化**```css.task-item-enter {  opacity: 0;  transform: translateX(-20px);}.task-item-enter-active {  transition: all 0.3s ease-out;}```3. **分类筛选**增加筛选按钮，根据完成状态过滤任务列表## 六、总结通过本案例，我们实践了以下核心技能：- DOM元素的动态创建与删除- 事件处理与委托机制- 数据持久化存储方案- 响应式数据绑定实现- 代码组织与模块化思想建议读者在此基础上尝试添加更多功能，例如：- 任务优先级设置- 拖拽排序支持- 夜间模式切换- 数据导入/导出功能DOM操作是前端开发的基石，熟练掌握这些技能将为学习现代前端框架打下坚实基础。希望这个案例能帮助大家更好地理解数据与视图的交互原理。
