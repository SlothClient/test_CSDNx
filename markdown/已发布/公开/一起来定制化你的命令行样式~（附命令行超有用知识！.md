# 设置
## 用户级别设置

 1. <kbd>win + r</kbd>键入`cmd`或者`powershell`，回车普通用户打开即可
 2. 点开顶部Tab加号旁边的下拉列表
 3. 找到设置，打开
 ![wizard1](https://pub-93427c0daded44f4ba9e82a03e799750.r2.dev/csdn-images/dc/dcfbc4896ca5cb9eca3e128f9a4836514a601d9910682dd5cd6a9d48d0c79e5e.png)
 4. 选择配色方案进行命令行统一设置（cmd,powershell,ubuntu...）
![wizard2](https://pub-93427c0daded44f4ba9e82a03e799750.r2.dev/csdn-images/82/8257bc2fae4fbf3a43e492b1e72f699d9559f1032d8b904908920587f37677d4.png)
 5. 选择一个喜欢的配色方案点击进去可以预览，我选择的是`One Half Dark`，然后选择设置为默认，这样就完成了统一设置
![wizard3](https://pub-93427c0daded44f4ba9e82a03e799750.r2.dev/csdn-images/39/39a5d535622ac87c87059c8a4c04abfea7e013bd99a9ac8c98a24010f01096d6.png)
 6. 进入默认值查看设置是否生效并进行详细设置
![wizard4](https://pub-93427c0daded44f4ba9e82a03e799750.r2.dev/csdn-images/9e/9eb72ebe83b87b6321517c258068dc08248ce4af909554d0e67ad1a9dc78e92b.png)
 7. 设置透明度和亚克力板效果

> 推荐开透明度，亚克力不建议开。
> 
> 有的时候照着网上找的文档打命令，但是总被命令行窗口挡住，这时候打开透明度就可以完美解决，我是开到了80%，完全可以看清。

 8. 所有命令行都一样太单调？不好区分？还可以分别设置，这里的优先级比默认高
![wizard5](https://pub-93427c0daded44f4ba9e82a03e799750.r2.dev/csdn-images/13/13eab9a2930c8e32698b0e0860537c9a2072f0c29a05563191f20dbcaa735661.png)
## 管理员级别设置

> 管理员权限打开的命令行和用户级别的不同，需要单独设置，比如拿我没改管理员权限样式的ubuntu命令行为例：
![用户权限](https://pub-93427c0daded44f4ba9e82a03e799750.r2.dev/csdn-images/8d/8d3fa85950c243cb53e9f93b69082ff6cfa4266e044865b75a0c11e79fef6aaa.png)
对了，我这里又改了一个外观设置和一个复古的设置，所以和刚开始看着不一样了
![管理员权限](https://pub-93427c0daded44f4ba9e82a03e799750.r2.dev/csdn-images/d5/d57d08f1878a72b4226f1bf5dcccca6321d1d68e7e7d8b0abbc3fe0a417d5a6d.png)
第一张是用户级别，使用的是我的全局默认设置
第二章是管理员级别，使用的是默认设置

 1. <kbd>win + r</kbd>键入`cmd`/`powershell`后<kbd>ctrl + shift + enter</kbd>管理员权限打开

 2. 点击管理员旁边的图标（这里有点难发现），选择属性打开属性面板
![wizard6](https://pub-93427c0daded44f4ba9e82a03e799750.r2.dev/csdn-images/25/255af8e31cf6692fad27aecdfc35bb1719796a58c9c06286d3f77790c32bfdcf.png)

 3. 进入属性面板框框一顿改，你怎么喜欢怎么来，我的建议是能看懂的就改，不能看懂的你也别瞎霍霍了。不过也不要看到管理员权限就不敢改了，看步骤二的图中下拉菜单，看到默认值了吗？你想改回去照着默认值改就行了。
![wizard7](https://pub-93427c0daded44f4ba9e82a03e799750.r2.dev/csdn-images/43/43fea4f9d4552d064bd77243ca785c4785588a419fd9040a2a413663b7c3105b.png)

## 其他你需要知道的设置

 1. 命令行中选中一段文字默认鼠标右击默认复制，空白处再次右击默认粘贴，外部复制的文本也一样，比`Ctrl C``Ctrl V`更简单的方式出现了🤣

> 在设置中修改右击行为也可以调出菜单，有复制、拆分窗格等功能（我以前一直以为是为了安全不准右键，有的时候复制都不能复制...

 1. 还在为命令行无法查找烦恼？怎么样能像vscode等软件内部的终端一样支持搜索？windows`Ctrl + F`,`Ctrl + Shift + F`调不出来？<kbd>Ctrl + Shift + P</kbd>打开命令面板，选择查找就可以了；虽然它有标识`Ctrl + Shift + F`，但我是真的调不出来。
![wizard8](https://pub-93427c0daded44f4ba9e82a03e799750.r2.dev/csdn-images/53/53f7819afe2df18280e94ba9697d259b5ed71f63a82586251916a9d50b398856.png)
 2. 同时开俩不同命令行跑命令
![wizard9](https://pub-93427c0daded44f4ba9e82a03e799750.r2.dev/csdn-images/09/091cc3256735d97f74ab13daf31646f34bfeb360c2aa7dc29675ce5c27bb84e2.png)
 4. 微软自己写的文档，自己去找吧..
