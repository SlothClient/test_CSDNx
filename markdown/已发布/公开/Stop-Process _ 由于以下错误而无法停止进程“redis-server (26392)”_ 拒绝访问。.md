> 基于[windows中使用cmd/powershell查杀进程](https://blog.csdn.net/weixin_73334344/article/details/151123329?spm=1011.2415.3001.5331)实践过程中遇到标题所示问题，故有此篇。

## 快速解决
进入cmd/powershell之前同时按住
<kbd>Ctrl</kbd>+<kbd>Shift</kbd>+<kbd>Enter</kbd>
打开命令行即可
***注意命令行显示的路径，不是用户而是system说明已是管理员身份***
