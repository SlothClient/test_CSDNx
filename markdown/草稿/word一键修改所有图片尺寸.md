# 一键修改Word中所有图片尺寸的方法
## 方法一：利用F4键快速调整
### 步骤1：调整首张图片尺寸
1. 在Word文档中选中需要调整的第一张图片
2. 通过以下两种方式进入尺寸设置界面：  
 - 【格式】菜单 → 【大小】组   
 - 右键菜单 → 【布局】→ 【大小】对话框
 3. 在设置界面输入具体的「高度」和「宽度」数值
 
### 步骤2：批量应用尺寸
1. 依次选中其他需要调整的图片
2. 按下键盘上的 `F4` 功能键（Mac用户需按 `Command+Y`）
3. 所有选中图片将自动应用首张图片的尺寸设置---## 方法二：通过自定义功能区批量操作### 步骤1：添加「选择多个对象」命令1. 点击【文件】→ 【选项】→ 【自定义功能区】2. 在右侧「主选项卡」区域：   - 选择「开始」选项卡   - 点击「新建组」按钮创建自定义分组3. 在左侧命令列表中找到「选择多个对象」功能4. 点击「添加」按钮将该功能移至新建的组### 步骤2：批量调整操作1. 点击新增的「选择多个对象」按钮2. 在弹出窗口中：   - 点击「全选」选中所有图片   - 可手动勾选/取消特定图片3. 确认选择后点击「确定」4. 在【格式】菜单的【大小】组中统一调整数值---## ⚠️ 注意事项1. **版本兼容性**：   - F4功能适用于Word 2010及以上版本   - 部分旧版本可能不支持批量选择功能2. **图片处理建议**：   - 批量操作前建议「取消勾选」：     ▢ 锁定纵横比       ▢ 相对原始图片大小   - 处理大量图片时可能出现短暂卡顿3. **操作技巧**：   - 按住 `Ctrl` 可进行多选操作   - 使用「布局」对话框可精确到0.01厘米   - 批量调整后可按 `Ctrl+Z` 分级撤销> **提示**：建议先对文档进行备份，再进行批量操作。若需保持图片比例，只需在首张图片调整时保持「锁定纵横比」的勾选状态。
